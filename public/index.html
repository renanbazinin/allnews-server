All News Server API

Welcome to the All News Server! This API aggregates news from various sources.

Available Endpoints:

/all-news
Get latest 50 news articles from all sources (international + Hebrew)
/all-news-heb
Get latest 50 news articles from Hebrew sources only

International Sources:

/bbc
BBC World News
/nyt
New York Times World News

Hebrew Sources:

/ynet
Ynet News
/maariv
Maariv News
